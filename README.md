# Bank Loan Default Analysis

## Project Overview
End-to-end loan default risk analysis on 143,983 
bank loans identifying key risk factors using 
Python, Excel and Power BI.

## Key Findings
- Overall default rate: 24.5%
- Type2 loans have highest default rate: 34.1%
- p2 loan purpose is riskiest at 33.0%
- l2 credit worthiness defaults at 31.8%
- Total high risk loans identified: 6,852
- Avg loan amount (default): $320,553
- Avg loan amount (non-default): $335,411

## Tools Used
- Python (Pandas, Seaborn, Matplotlib, Scipy)
- Excel (KPIs, Pivot Tables)
- Power BI (Interactive Dashboard)

## Dataset
Kaggle: Bank Loan Default Dataset
143,983 loans | 36 features

## Workflow
1. Downloaded raw data from Kaggle
2. Cleaned 181,135 null values in Jupyter
3. Engineered HighRisk and LoanAmountGroup features
4. Built KPIs and Pivot Tables in Excel
5. Created interactive dashboard in Power BI
